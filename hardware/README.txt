usb2ax Eagle files
==================

usb2ax_v3.2 : Second production version, cheaper (uses OPL parts instead of some costly components).

usb2ax_v3.1 : First production version. Added protection against short-circuit between +12V and DATA, and better hackability (expension pads).

usb2ax_mini : A smaller version of the v3.0 with a PCB USB connector and audacious placement of the Dynamixel connector.

usb2ax_v3.0 : MCU in QFN package, real USB connector

usb2ax_v2 (removed) : aborted version similar to v1.

usb2ax_v1 : Initial realease. MCu in TQFP package, PCB USB connector, protections lacking.
